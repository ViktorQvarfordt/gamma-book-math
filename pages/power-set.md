# Power set

:::definition[Power set]
The power set $\mathcal{P}(A)$ of a [set]() $A$ is the set of all [subsets](subset) of $A$.
:::

:::example
- $A = \{1, 2\}$
- $\mathcal{P}(A) = \big\{\{\}, \{1\}, \{2\}, \{1,2\}\big\}$
:::

:::example
- $A = \{1, 2, 3\}$
- $\mathcal{P}(A) = \big\{\{\}, \{1\}, \{2\}, \{3\}, \{1,2\}, \{2,3\}, \{1,3\}, \{1,2,3\}\big\}$
:::
